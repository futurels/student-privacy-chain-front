**文档性质：** 统一接口设计说明书  
**适用范围：** 后端开发、前端联调、Swagger 整理、论文附录

## 1. 设计约定

### 1.1 基础路径
- 建议统一前缀：`/api`
- 建议响应编码：`UTF-8`
- 默认请求/响应格式：`application/json`
- 文件上传接口采用：`multipart/form-data`

### 1.2 认证方式
- 除登录、刷新令牌等少量接口外，其余接口默认需要登录后访问
- 建议请求头：`Authorization: Bearer <accessToken>`

### 1.3 通用返回结构
```json
{
  "code": 0,
  "message": "success",
  "data": {},
  "traceId": "TRX202604060000",
  "timestamp": "2026-04-06T00:00:00+08:00"
}
```

### 1.4 通用返回字段说明
| 字段 | 类型 | 说明 |
|---|---|---|
| code | integer | 业务状态码，0 表示成功 |
| message | string | 响应描述信息 |
| data | object | 业务数据载荷，不同接口结构不同 |
| traceId | string | 链路追踪标识 |
| timestamp | string | 服务端响应时间，ISO 8601 |

### 1.5 分页返回说明
列表接口的 `data` 建议统一包含以下字段：

| 字段 | 类型 | 说明 |
|---|---|---|
| records | array | 当前页记录集合 |
| total | integer | 总记录数 |
| pageNum | integer | 当前页码 |
| pageSize | integer | 每页条数 |

### 1.6 常用错误码
| 错误码 | 含义 |
|---|---|
| 0 | 成功 |
| 400 | 请求参数错误或格式校验失败 |
| 401 | 未登录或令牌失效 |
| 403 | 无权限访问当前资源 |
| 404 | 目标资源不存在 |
| 409 | 业务状态冲突，例如重复申请、重复审批 |
| 422 | 业务规则校验失败，例如授权已过期、申请状态不允许 |
| 500 | 系统内部错误 |
| 5001 | 区块链调用失败 |
| 5002 | IPFS 存储失败 |
| 5003 | 文件存储或加密处理失败 |

## 认证与账户

### API-01 用户登录

| 项目 | 内容 |
|---|---|
| 接口名称 | 用户登录 |
| 所属模块 | 认证与账户 |
| 请求方法 | POST |
| 请求路径 | `/api/auth/login` |
| 调用角色 | 全部角色 |
| 优先级 | P0 |
| 接口说明 | 用户使用学号/工号或账号密码登录系统并获取访问凭证。 |

#### 请求参数

| 参数名 | 位置 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| username | body | string | 是 | 登录账号，学生可为学号，管理员可为工号或系统账号 |
| password | body | string | 是 | 登录密码 |
| loginType | body | string | 否 | 登录类型，默认为 PASSWORD |

#### data 字段说明

| 字段 | 类型 | 说明 |
|---|---|---|
| accessToken | string | 访问令牌 |
| tokenType | string | 令牌类型，建议为 Bearer |
| expiresIn | integer | 令牌有效期，单位秒 |
| userInfo | object | 当前登录用户简要信息，包含 id、name、roleCodes、departmentName |

#### 请求示例

```json
{
  "username": "2205110212",
  "password": "******",
  "loginType": "PASSWORD"
}
```

#### 响应示例

```json
{
  "code": 0,
  "message": "登录成功",
  "data": {
    "accessToken": "eyJhbGciOi...",
    "tokenType": "Bearer",
    "expiresIn": 7200,
    "userInfo": {
      "id": 1001,
      "name": "梁爽",
      "roleCodes": [
        "STUDENT"
      ],
      "departmentName": "软件工程学院"
    }
  },
  "traceId": "TRX202604060001",
  "timestamp": "2026-04-06T10:00:00+08:00"
}
```
